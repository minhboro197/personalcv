Automatic Sorting Trashbin

Description

Different from normal trash bins, there are sensors, motors, microcontroller(Arduino) integrated into this trash bin to help it sorts garbage. The entire height of this design is 130 cm, with the perimeter is about 100cm

When garbage is thrown into the tunnel, there are some sensors, which scan the object and send the signal to the Arduino, then Arduino will identify and drop it to the right bins by rotating the tunnel
